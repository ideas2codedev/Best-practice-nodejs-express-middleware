## Requirements
-   Node >= 8.10
-   npm >= 5.6

## Installation
**Clone**  this repository
```
git https://github.com/ideas2codedev/Best-practice-nodejs-express-middleware.git
```
Move to  **root folder**  and run:
```
npm install
```
Now let's start our application run:
```
npm start
```
